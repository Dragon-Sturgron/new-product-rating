# 更新说明：删除弹框美化 + 款式 Excel 导入

本版本在上一版基础上新增：

1. 后台所有原生删除确认框改为自定义美化弹框。
   - 单个款式删除
   - 已配置款式全部删除
   - 单次评分结果删除
   - 评分结果全部删除

2. “已配置款式”新增“导入款式”按钮。
   - 支持 `.xls`、`.xlsx`
   - 读取字段：`款式编码`、`季节`、`基本售价`
   - 款式编码已存在：更新季节和基本售价，保留原图片、备注和启用状态
   - 款式编码不存在：新增款式，默认启用评分，图片留空
   - 导入前会显示预览确认弹框

3. Excel 解析在浏览器端完成，会自动加载 SheetJS 解析库；如果网络无法加载，会在后台提示“Excel 解析库加载失败”。

## Excel 表头要求

第一行建议使用：

| 款式编码 | 季节 | 基本售价 |
|---|---|---|
| XA2408A | 秋冬 | 138 |

也兼容部分常见表头：款号、款式编号、编码、SKU、季度、售价、价格、吊牌价等。


## 本次更新：新增产品图复制粘贴上传

- “新增评分款式”的产品图上传区域新增复制粘贴模式。
- 支持三种方式：点击选择、拖拽图片、复制图片后按 Ctrl+V 粘贴。
- 粘贴图片后仍然只做浏览器本地预览，不会立即上传七牛云/OSS；点击“保存款式”后才会真正上传。

## 本次更新：导入款式弹框增强

- “已配置款式”的“导入款式”按钮改为打开弹框，不再直接弹出文件选择。
- 弹框支持点击选择、拖拽上传 `.xls/.xlsx`、复制文件或从 Excel 复制三列表格后粘贴。
- 弹框提供“确定 / 导入模板 / 取消”三个按钮。
- 新增导入模板文件：`public/assets/templates/style-import-template.xlsx`。


## 2026-07-20 导出与清空数据更新

- 将“导出评分 CSV”改为“导出评分 XLSX”，导出文件名为“评分结果.xlsx”。
- 新增顶部“清空全部数据”按钮，可一键删除“已配置款式”和“评分结果”的所有内容，并尝试同步清理款式图片。
- 导出 XLSX 使用内置 OpenXML 生成，不依赖外部服务，兼容 Excel/WPS。

## 2026-07-20 评分链接功能

新增“评分链接”能力：

- “款式配置 → 已配置款式”左侧新增勾选框，可勾选多个款式后生成独立评分链接。
- 链接路径为 `域名/随机码`，例如 `https://example.com/a8K29mQx`。
- 生成时可设置链接名称、有效期、备注。
- 后台新增“评分链接管理”页，可查看、复制、打开、删除评分链接。
- 评分链接到期后不会自动删除记录，但前端无法访问、无法提交。
- 评分人通过链接进入后，只能看到该链接包含的款式。

## 2026-07-20：评分链接改为必需

- 直接访问域名首页 `/` 不再进入评分系统，改为提示“访问地址有问题，请联系管理员”。
- 只有后台“评分链接”中生成且未删除、未停用、未过期的链接可以进入评分。
- 关闭无链接读取全部款式的 `/api/public/styles` 接口。
- 批量提交评分和评分草稿都必须携带有效评分链接编码。
- 关闭不受评分链接约束的旧版单条评分接口 `/api/public/scores`。
- `functions` 与 `edge-functions` 已同步修改，可继续部署到 EdgeOne Pages。

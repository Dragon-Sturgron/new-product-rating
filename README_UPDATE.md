# 更新说明：删除弹框美化 + 款式 Excel 导入

本版本在上一版基础上新增：

1. 后台所有原生删除确认框改为自定义美化弹框。
   - 单个款式删除
   - 已配置款式全部删除
   - 单次评分结果删除
   - 评分结果全部删除

2. “已配置款式”新增“导入款式”按钮。
   - 支持 `.xls`、`.xlsx`
   - 读取字段：`款式编码`、`季节`、`基本售价`
   - 款式编码已存在：更新季节和基本售价，保留原图片、备注和启用状态
   - 款式编码不存在：新增款式，默认启用评分，图片留空
   - 导入前会显示预览确认弹框

3. Excel 解析在浏览器端完成，会自动加载 SheetJS 解析库；如果网络无法加载，会在后台提示“Excel 解析库加载失败”。

## Excel 表头要求

第一行建议使用：

| 款式编码 | 季节 | 基本售价 |
|---|---|---|
| XA2408A | 秋冬 | 138 |

也兼容部分常见表头：款号、款式编号、编码、SKU、季度、售价、价格、吊牌价等。


## 本次更新：新增产品图复制粘贴上传

- “新增评分款式”的产品图上传区域新增复制粘贴模式。
- 支持三种方式：点击选择、拖拽图片、复制图片后按 Ctrl+V 粘贴。
- 粘贴图片后仍然只做浏览器本地预览，不会立即上传七牛云/OSS；点击“保存款式”后才会真正上传。

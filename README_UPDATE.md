# 更新说明：评分结果只读 + 后台设置国内 OSS

本版本调整：

1. 后台“评分结果”只允许查看。
   - 列表操作只保留“查看/收起”。
   - 明细里不再显示编辑、历史、删除按钮。
   - 后端 `/api/scores/:id` 的 PUT/DELETE 会返回只读提示，避免旧缓存 JS 继续修改数据。

2. 图片存储配置集中到后台“设置 → 图片存储配置”。
   - 不再需要通过环境变量配置七牛云/阿里云 OSS/腾讯云 COS 参数。
   - 设置页新增“国内 OSS 服务商”选择，可选择七牛云、阿里云 OSS、腾讯云 COS、MinIO/其他 S3。
   - 选择服务商后会辅助填写常见 Endpoint/Region/Path Style，实际 Bucket、AccessKey、SecretKey、公开访问域名仍在页面里填写。

EdgeOne 仍需保留：

```text
STORAGE_DRIVER=edgeone-kv
EDGEONE_KV_NAMESPACE=product_review
```

KV 绑定仍保持：

```text
变量名：EDGEONE_KV
命名空间：product_review
```


## 2026-07-10 更新：评分等级说明和区间后台配置
- 后台“设置”新增“评分等级说明”。
- 可自定义前端顶部说明文字。
- 可配置大单/中单/小单试水/建议不下等等级名称和最低百分比区间。
- 每个评分体系按“该体系得分 ÷ 该体系满分 × 100%”单独判断等级。
- 未完成草稿和最终提交逻辑不变。


## 2026-07-10 更新：移除评分等级说明配置区域
- 按要求删除后台“设置”页中的评分等级说明/区间配置区域。
- 保留现有评分项、评分类型、图片存储等配置，不影响 EdgeOne KV 数据。


## 本次更新：评分结果支持删除

- 后台评分结果仍然不允许编辑修改。
- 每条提交记录增加“删除”按钮，可删除该评分人本次提交的所有款式评分。
- 删除后评分结果列表不再显示该提交记录。
- EdgeOne KV 下会同步刷新当天提交标记；如果该评分人当天记录被全部删除，可重新提交当天评分。


## 2026-07-17 更新：已配置款式支持全部删除

- 后台“已配置款式”区域新增“全部删除”按钮。
- 删除范围为当前查询条件下的款式；没有筛选条件时删除全部已配置款式。
- 删除后不可恢复，前端评分页不会再显示被删除款式。
- 评分结果的全部删除功能保持不变。


### 七牛云公开访问路径前缀

如果七牛云实际图片地址类似：

```text
http://ti9dkt322.hn-bkt.clouddn.com/xianglupiju/review-images-xxx.png
```

后台“图片公开访问域名 / CDN域名”填写自定义域名，例如：

```text
https://xianglu.dragon-sturgeon.cn
```

同时“公开访问路径前缀”填写：

```text
xianglupiju
```

系统最终会拼接为：

```text
https://xianglu.dragon-sturgeon.cn/xianglupiju/review-images-xxx.png
```

如果你的自定义 CDN 域名已经直接绑定到空间根路径，不需要 `/xianglupiju`，则路径前缀留空。

## 2026-07-18 更新：删除/更换款式图片时同步清理 OSS

- 修改“已配置款式”的产品图时，如果旧图片是本系统上传到当前配置的 R2/S3/七牛/OSS 中的图片，会自动调用对象存储删除旧图片。
- 删除单个款式时，会同步尝试删除该款绑定的产品图。
- “已配置款式 → 全部删除”会批量删除款式，并批量尝试删除对应产品图。
- 为了防止误删，只会删除能从当前图片存储配置中识别出来、且符合当前文件名前缀的图片；手动粘贴的外部图片链接不会被删除。
- 如果 OSS 删除失败，款式删除/更新仍然会完成，接口会返回 image_delete / image_delete_results 便于排查。
